Loops - There are several types of looping statements supported in JavaScript. These types are: while, do/while, for, and for/in


While loop example:
This example counts from 1 to 10 and appends the number to a variable that is comma delimited.
The example also uses a conditional "if" statement to decide when to use the comma delimiter.

Code Snippit

	function countToTen(){
		var exampleText="While Ouput: ";
		var iterator = 1;
		var x = document.getElementById("whileOutput")
		while (iterator <= 10) {
			exampleText+= iterator;
			if (iterator != 10) {
				exampleText+= ", ";
			}
			iterator++;
		}
		x.innerHTML = exampleText;
	}

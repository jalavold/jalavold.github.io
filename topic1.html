Example 1: "while()" loop and "if()"

This example counts from 1 to 10 and appends the number to a variable that is comma delimited.
The example also uses a conditional "if" statement to decide when to use the comma delimiter.

Code Snippit

	function countToTen(){
		var exampleText="Example 1 Ouput: ";
		var iterator = 1;
		var x = document.getElementById("ex1Output")
		while (iterator <= 10) {
			exampleText+= iterator;
			if (iterator != 10) {
				exampleText+= ", ";
			}
			iterator++;
		}
		x.innerHTML = exampleText;
	}

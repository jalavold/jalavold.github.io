Topic 4 - Using XMLHTTPRequest to Consume a JSON Web Service


This table of users was created by querying a JSON webservice and retriving the returned data via XMLHttpRequest(). The JSON is then parsed into a JavaScript object using the JSON.parse() JavaScript Function. Then iterating through the items in the JavaScript object and generating the necessary HTML output.

Sample JavaScript Code that implements XMLHttpRequest(), JSON.parse(), and JSON.stringify()


    <script>
        // Using XMLHttpRequest() to get the movies.json file from the server. 
        // Once the readyState is 4 (finished and response ready) and the status is 200 (OK).
        // Next the responseText is parsed using JSON.parse into a JavaScript object.  
        // Finally the object is passed by value as a parameter to the writehtml function.
        var xmlhttp = new XMLHttpRequest();
        if ('withCredentials' in xmlhttp){
            xmlhttp.onreadystatechange = function () {
            if (this.readyState == 4 && this.status == 200) {
                var jsonDataObj = JSON.parse(this.responseText);
                writehtml(jsonDataObj);
                }
            };
            xmlhttp.open("GET", "https://jsonplaceholder.typicode.com/users", true);
            xmlhttp.send();
        }
        
         
        // This function takes the ojbects value passed to it and generates the page's additional HTML.
        // Nested for in loops are used to access the values of each key/value pair and generates the table HTML.
        function writehtml (jsonDataObj){
            var htmltable = "<table>";
            htmltable += "<tr><th>User Name</th><th>Email Address</th><th>Phone Number</th></tr>";
            for (var inc = 0 ; inc < jsonDataObj.length ; inc++){
                htmltable += "<tr><td>" + jsonDataObj[inc].name + "</td><td>" + jsonDataObj[inc].email + "</td><td>" + jsonDataObj[inc].phone + "</td></tr>";
            }
            htmltable += "</table>";
            document.getElementById("jsonstringify").innerHTML = JSON.stringify(jsonDataObj);
            document.getElementById("movieTable").innerHTML = htmltable;
        }
    </script>

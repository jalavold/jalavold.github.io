There a several ways you create an object.

  1. Create an empty object "var myobj = {};" and then add the key/value pairs using one of the methods below.
    1. Dot notation. Specifiy object variable then a dot "." and the key/property name. Then make that equal to a value. i.e. myobj.key1 = "Booya";
    2. Bracket notation. Specify oject variable appending ["property"] and then make that equal to a value. i.e. myobj["key1"] = "Booya";
  2. Create an objest using Object literal method. Object is created and defined in a single statment. It consists of list of name:value pairs. i.e. "var car2 = {owner:"Lavold Family", make:"Honda", model:"Civic", year:"2013"};"
  3. Create an object using a user defined constructor. One thing to keep in mind is that unlike with standard JavaScript objects, you should use the "new" keyword when creating and object from a user defined constructor. Unless you want to make a boolean, Number, or String an object.
    • Example:  var aaronsCar = new Car("Aaron","Jeep","Rubicon","2018");

You can create your own object type by defining an object constructor. Below is more info about how to define and use a constructor.

  1. Best practice is to use an upper-case letter for the first letter in the function name.
  2. This is an example of definine a object constructor 
    •             
            //  Object constructor prototype for Car object
            function Car(owner, make, model, year){
                this.carOwner = owner;
                this.carMake = make;
                this.carModel = model;
                this.carYear = year;
            }
  3. You can add to an existing contructor by using the prototype property. The example below adds a method to the existing Car object constructor.
    •             
            // adding a method to the Car prototype could also add a property like this if desired
            Car.prototype.fullDescription = function() {
                return this.carYear + " " + this.carMake + " " + this.carModel;
            }
